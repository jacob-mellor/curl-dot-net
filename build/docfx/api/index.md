# API Reference

This is the complete API reference documentation for CurlDotNet.

## Namespaces

- **CurlDotNet** - Main namespace containing the Curl class and core functionality
- **CurlDotNet.Core** - Core classes like CurlOptions, CurlResult, and CurlRequestBuilder
- **CurlDotNet.Exceptions** - All exception types thrown by CurlDotNet
- **CurlDotNet.Middleware** - Middleware components for request/response pipeline
- **CurlDotNet.Extensions** - Extension methods for fluent API
- **CurlDotNet.Lib** - Low-level LibCurl API access

## Key Classes

### Curl
The main entry point for executing curl commands.

### CurlOptions
Configuration options for curl requests.

### CurlResult
The result object returned from all curl operations.

### CurlRequestBuilder
Fluent API for building curl requests.

### LibCurl
Low-level access to curl functionality.
